在Eclipse下编辑XML文件时可以像Java文件一样进行格式化。
不过，默认的格式化设置效果不太理想，尤其是标签属性较多时。

通过以下设置可以达到一个相对理想的效果：
Window -> Preferences -> XML -> Editor -> Formatting :

设置每行宽度
Line width
设定为80到100个字符。

选中
Split multiple attributes each on a new line
这样标签的每个属性都单独一行显示。

不选中
Align final bracket in multi-line element tags
这样，标签结束符不单独占用一行。

选中
Clear all blank lines
去掉所有空行。

选中
Insert whitespace before closing empty end-tags
在标签结束符前添加空格。

选中
Indent using spaces
缩进使用空格代替tab。

设定缩进宽度
Indentation size
设定为4，即缩进4个空格。
