今天上午我发过求租贴，帮助的朋友不少，但是还是没有到点子上，现在我已经成功解决这个安装问题。同时游戏也能正常进入。

现把方法提供出现，以便出现同样问题的朋友使用。

问题出在自动安装Microsoft Visual C++ 2005 SP1，就是光盘文件中的vcredist_x86这个文件，报错

出错信息“command line option syntax error,type command/? for help”。

解决方面还是要安装上这个2005版本，我试了安装2008版本，也不能进入游戏。方法如下

用Winrar软件点中vcredist_x86这个文件，解压缩，得到VCREDI~3这个文件，然后再解压缩

等到vcredist和vcredis1，运行前者，然后就开始安装Microsoft Visual C++ 2005 ，系统不会有任何提升是否安装好，只需去"添加或删除程序“中看下Microsoft Visual C++ 2005 是否已经在里面就可以了

随后正常进入游戏，一切正常。
